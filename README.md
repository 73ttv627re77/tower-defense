# tower-defense

Stylized 2D top-down tower defence game for iOS. Overgrown stone ruins theme. 100 hand-designed levels, 8 upgradable tower types, 15 enemy types. Prototype first, full campaign after.

## Status

**Phase 1 (prototype) in progress.** 1 map, 5 levels, 3 towers, 5 enemies, runnable on iPhone 17 Pro simulator.

See the [GitHub Issues](https://github.com/73ttv627re77/tower-defense/issues) for active work.
